# 네이버쇼핑 상품검색 SEO — 업로드 강조사항

> **출처**: [상품검색SEO FAQ (H00015)](https://join.shopping.naver.com/faq/list.nhn?catgCd=H00015)  
> **정본 설정**: `config/listing.yaml` → `naver_shopping_seo`  
> **코드 적용**: `src/render/seo_rules.py` · `templates/marketplace_detail.j2` · `Copywriter`

네이버쇼핑 검색 SEO는 **가격비교·N+스토어(스마트스토어) 공통** 가이드입니다.

## 검색 랭킹 3요소

| 요소 | 의미 | listing-forge 대응 |
|------|------|-------------------|
| **적합도** | 검색어 ↔ 상품명·브랜드·카테고리·속성·태그 연관도 | `Copywriter` 상품명 · `coupang_meta` · CSV 검색키워드 |
| **인기도** | 클릭·판매·구매평·찜·최신성 | 파이프라인 밖 (운영 데이터) |
| **신뢰도** | SEO 준수·페널티·어뷰징 | `NaverShoppingSeoRules` 상품명 정규화 |

## 상품명 — 반드시 지킬 것

1. **공식 정보만** — 브랜드·시리즈·모델명·대표 옵션(색상/용량/사이즈) 순, **50자 내외** 권장 (최대 100자)
2. **금지** — 무료배송·할인·특가·1위·정품·이벤트 등 **판매/홍보 문구** (→ 이벤트 필드)
3. **금지** — 동의어·유의어 **중복 나열** (어뷰징 판정)
4. **금지** — 셀러명·쇼핑몰명·카테고리 키워드 stuffing
5. **금지** — ★▶◀ 등 **비허용 특수문자** ( `( ) - · [ ] / & + , ~ .` 만 최소 사용)
6. **TIP** — 대표 옵션 1개·연관 키워드 **1~2개**만 상품명에 포함
7. **TIP** — 띄어쓰기 적절히 (과도한 붙임/띄움 금지)

## 상품 정보 필드

| 필드 | 강조 |
|------|------|
| 브랜드/제조사 | 빠짐없이 정확히 |
| 카테고리 | 상품과 일치하는 최하위 카테고리 |
| 속성·태그 | 검색 가능 정보 최대 제공 (태그 20개 등) |
| 이벤트 필드 | 할인·배송·사은품은 **여기만** (상품명·상세 본문 X) |

## 상품 이미지

- **500×500px 이상** (JPG, 4MB 미만), 대표 1 + 추가 2장 이상
- **흰/단색 배경** 대표컷 권장 (쿠팡 `BackgroundNormalizer`와 동일 원칙)
- **금지** — 과도한 텍스트·워터마크·흐린/과보정 이미지
- 상세 HTML: **max-width 860px**, `alt`에 상품명·캡션

## 상세 HTML (쿠팡·네이버 동일)

`build` 시 `naver/` · `coupang/` 모두에 **동일** `detail_placeholders.html` · `detail_final.html` 생성.  
이미지 경로: `../naver-store-images/` (공통 store-images 폴더 참조).

## 하지 말 것 (페널티)

- 상품 **재등록으로 최신성**만 노리기
- 키워드 stuffing·동의어 반복
- 상품명 SEO 위반 수정 (수정 자체는 감점 없음, **위반 내용**은 불이익)
